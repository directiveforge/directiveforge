## Active servers

### context7 — `@upstash/context7-mcp@3.2.2` (npm, pinned)

- **Posture**: RO — fetches public documentation only; no credentials
- **Add when**: always — prevents outdated-API hallucinations for FastAPI 0.115 / Pydantic v2 / SQLAlchemy 2.0 (all frequently hallucinated as older APIs)
- **Skip when**: fully offline environments

### postgres — `mcp-postgresql-ops@3.2.8` (PyPI, run via `uvx`, pinned)

- **Posture**: RO — 30+ read-only DBA tools (schema inspection, EXPLAIN, index/table stats, vacuum monitoring); no write surface. Connection string via `${DATABASE_URL}` — never a literal value. Use a read-only DB user in production for defense in depth.
- **Add when**: agents need to understand the Postgres schema or verify/EXPLAIN a query
- **Credentials**: `DATABASE_URL` (read-only user recommended). For local SQLite dev this server is Postgres-oriented — point it at the Postgres instance, not the SQLite file
- **Why this package and NOT `@modelcontextprotocol/server-postgres`**: the npm `@modelcontextprotocol/server-postgres` was archived May 2025 and has a known SQL-injection advisory (Datadog Security Labs) — **DO NOT USE**. `mcp-postgresql-ops` is the vetted registry replacement. See `<KIT_ROOT>/knowledge-base/MCP-SERVER-REGISTRY.md`.

## Security protocol

- Pin npm/stdio/uvx server versions (`@x.y.z`, never `@latest`); remote HTTP servers are version-managed upstream
- `${ENV_VAR}` for every credential — literal values in config are a checklist §5 failure
- After any MCP config change: `uvx snyk-agent-scan@latest` (full verdict needs `SNYK_TOKEN`; without it it degrades to inventory and auto-declines stdio servers non-interactively)
- Registry of vetted servers + postures: `<KIT_ROOT>/knowledge-base/MCP-SERVER-REGISTRY.md`

## Appendix — `.claude/settings.json` levers

Semantics per `<KIT_ROOT>/workflows/WORKFLOW-CLAUDE-CODE.md` §7.4 / §7.5 / §11.3:

- `MAX_THINKING_TOKENS` — set to `10000` here for routine maintenance work; `0` disables thinking entirely
- `CLAUDE_AUTOCOMPACT_PCT_OVERRIDE` — `50` for long sessions; compacts before the quality cliff
- `CLAUDE_CODE_SUBAGENT_MODEL` — **omitted** so subagents inherit the main model (v2.1.196+ inherit-on-omit)
- Hook `PostToolUse Write|Edit` — `hooks` is `{}` (no formatter is configured in this project; nothing to run on write)
- Permissions: pre-allow safe reads + the real gates (git reads, `python -c "import app.main"`, `alembic upgrade/revision/current/history`, `uvicorn`, `pip install`); deny destructive (force-push, hard reset, `rm -rf`, `.env` reads)
