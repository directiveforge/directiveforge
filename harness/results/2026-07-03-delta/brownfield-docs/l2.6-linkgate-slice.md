## Gate design (root-cause check)

`check-links.py:49` resolves each link against `os.path.dirname(fpath)` — the **linking file's own directory**, not the cwd/repo root. This is exactly the fix the baseline C1 defect demanded. Supporting behavior:

- `strip_code()` (lines 21–35) drops fenced (```` ``` ````/`~~~`) and inline `` `code` `` spans before link extraction, keeping line numbers aligned.
- External schemes (`http(s):`, `mailto:`, `tel:`, protocol-relative `//`) skipped (line 52).
- Pure `#anchor` self-links skipped; `#frag`/`?query` stripped before existence test (lines 54–57).
- `EXCLUDE` args prune both dirs and files by normalized-path prefix (lines 15–19).

`quality-gates.md` pins the invocation: `python3 .claude/scripts/check-links.py docs/archive` — archive excluded because frozen reports may point at superseded siblings.
